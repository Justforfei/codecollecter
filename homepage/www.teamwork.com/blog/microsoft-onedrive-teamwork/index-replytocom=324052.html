Microsoft OneDrive comes to Teamwork

Posted August 29, 2014

Connecting external cloud storage to Teamwork Project Manager has always been one of the platform’s most popular features, and we’re delighted to have another option for our users as we add Microsoft OneDrive integration to Teamwork.com.

You can find full details on how to integrate OneDrive in the video below.

You’ll also find an FAQ on the new OneDrive integration here.

Please note that because of how Microsoft’s OneDrive API works, this integration currently only works for the standard OneDrive account. Microsoft OneDrive for Business isn’t supported yet, but we’ll continue working on finding a solution to this.
