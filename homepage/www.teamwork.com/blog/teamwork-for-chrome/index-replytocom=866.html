Teamwork for Chrome

Posted January 31, 2014

Here at Teamwork.com we love Google Chrome, it has been our browser of choice for a long time. We noticed that we’re not alone, many of our customers also favour this browser, and today we are happy to announce the release of our Chrome extension – Teamwork for Chrome.

Key Features

  • Save the pages you’re visiting as links

  • Create new messages from anywhere in Chrome

  • Create tasks from anywhere in Chrome

  • Log time quickly from anywhere in Chrome

  • Easily switch between different Teamwork accounts

  • Custom Teamwork button for Gmail

 

Getting Started

Getting started is easy — install our extension on the Chrome Web Store, navigate to any page and add your account with your API key. For this to work, all you need to do is click the Teamwork icon in your Extensions toolbar and click Settings.

If you ever forget how it’s done – don’t worry, we’ve got the instructions built into the extension.

Note: Any open tabs will need to be reloaded after installation for the extension to work on those tabs correctly.

 

Work faster from Gmail

Teamwork for Chrome adds a special button to your Gmail toolbar when you open an email. You can now quickly add tasks, messages and time entries without having to switch tabs and leave Gmail!

Save links

See a website you like? Saving a link to it on Teamwork.com has never been easier! Click the Teamwork icon in your Extensions toolbar or right click anywhere on the page, click Create link and you can add it to the project of your choice.

 

Log time anywhere

Quickly log time on any page you visit! You can enter the time as 1.5 or 1:30. We know they both mean 1 hour and 30 minutes.

More tips

  • Switch between different Teamwork.com accounts in each form by clicking your name at the top of the form and choosing a different account.

  • Select a piece of text on a web page, right click (or Control-Click on Mac), navigate to Teamwork for Chrome and create a task or message. The text will be pasted into the task description or message body.

  • You can click Select Text for any description or message content to select text on the page to be pasted into the appropriate field.

  • Create new Message and Link categories on the go. Just click Create New, choose a name, click Create and it’s done!

 

We hope you find this article helpful. Install Teamwork for Chrome and speed up your workflow!
