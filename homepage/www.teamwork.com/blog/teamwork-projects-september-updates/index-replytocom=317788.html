Teamwork Projects – September Updates

Posted September 30, 2015

The Teamwork Projects team has been hard at work again, this month trying to find ways to improve your favorite project management tool. This month, we’re proud to announce loads of UI, general updates and mobile updates.

UI Updates

We’ve made massive improvements to our UI this month, making Teamwork Projects more beautiful and easier to use than ever! I know what you’re thinking… How can you improve on perfection? Well, our developers found a way!

We have listed Activity at Dashboard and Project level. This is now consistent across the board with Teamwork Desk and Teamwork Chat, making the integration between your three favorite products feel even more seamless. Now, it’s so much easier for you to consume information when you log into your account. You have a snapshot of what your team is up to and you can hit the ground running with your own work. Having Activity listed in the Dashboard will speed up the loading process too, so you can save time doing the things that matter – completing those tasks!

We’ve created an intuitive place for Task Lists to expand and collapse when viewing single lists (it’s the little – or + sign next to the name of the Task List).  Don’t waste time scanning through tasks that don’t concern you to find what it is you’re looking for. Filter out the noise and collapse Task Lists that you don’t need. Find tasks quicker and easier using this function. We’ve also put in a new options menu (the little drop-down icon) to give you more of a choice of action when you are looking to work on a task. This will make it easier for you to view the Task Lists that you’re working on and will make accessing options easier too.

Everything about the UI update is faster and more intuitive to make using the software even simpler for you and your team.

Some General Updates

  • We’ve added a new column to the task importer: “Status”. You can now import your ‘active’ or ‘completed’ tasks.
  • You can now filter completed tasks in Everything with tags.
  • You can now add tags to titles in Milestone Calendar.
  • When assigning tasks, if you double click on “Who should do this?” when you are already assigned, it switches back to “anyone.”
  • We have removed the paste re-size in comments and messages to improve image quality.

Mobile Updates

There were loads of fixes and small improvements for both Android and iOS. Both apps should be faster and more agile than ever! Last month, we introduced support for Likes and Multiple File Uploads. This month, the big updates are support for Risks, Likes, Notifications, and Multiple File Uploads for sharing extension (for iPhone).

In an update that will be rolling out in the next few days, we’ve also added a Today Extension for iPhone. You can complete tasks, start a timer, log time, and you can even comment on tasks all directly from the today section on your lock screen. It couldn’t be easier to set up or use. Check out this video to get up and running right away.

As you can see, it’s been business a usual here at Teamwork.com. If you have any questions about any of the updates mentioned in this blog, don’t hesitate to leave us a message here or e-mail our support team at suppport@teamwork.com.

Don’t forget to tune-in next month for a whole new list of features!
