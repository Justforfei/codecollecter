Aves UI is a large user interface kit containing hundreds of web elements, that can help you design entire websites in Photoshop with ease. Sets of icons, swatches and actions are included in the kit.

Components

―

The UI kit has been organized in several contextual components - widgets, navigation, forms, media, articles, UI elements and typography.

TYPOGRAPHY

―

Free fonts Lato and Heuristica are used throughout the kit.

They are designed for legibility across web and mobile interfaces and support several character sets, a wide array of supported languages, old style numerals and ligatures.

  • COLOR PALETTE

    ―

    Don’t bother choosing the right color. A 12-color palette comes ready to use in the UI kit. The palette is available as a PSD and also as a Photoshop Swatches file.

Icons

―

As a bonus, you get a set of outlined icons available in AI and PSD formats. The icons were built on a 16x16 grid, which allows you to easily resize them to 32x32, 64x64, 128x128 etc. They are perfectly aligned to a pixel grid and you can also customize outline strokes.

  • BUILD ENTIRE WEBSITES IN MINUTES

    ―

    With the choice of elements offered and the organized layer structure, building a website is just a matter of combining layer groups.
