February 2014 Updates and Fixes

Posted February 28, 2014

As always, we have been hard at work and even though it’s a shorter month, we’ve rolled out a lot of subtle yet important improvements to Teamwork.com. For those of you who are interested, the full list is available to view straight from our SVN Repository on our website.

Updates and Features

  • Update: Milestones: Clicking “Done” with CTRL pressed allows for comment to be added
  • Update: Tasks: Clicking checkbox to complete with CTRL Key pressed marks task as complete and opens the New Comment dialog
  • Update: Comments: Autosave now added to comments
  • File viewer: Removed excess close button
  • Likes: Removed icon and change likes to liked
  • Lightbox – confirm popping up multiple times.
  • Sub Tasks: Show more link on parent task view page
  • Task list: No activity log for list completion when list is pinned
  • ui:Updated lightbox header style

  • UI:New happiness.teamwork.com design implemented

  • Gantt: Disable all keyboard shortcuts except escape key press

Adrian Kerr June 10th, 2015

Hi Ketul,

We do have an option, it works based on the company you are a part of. All people in the owner company can by default see the people section and the contact details. For people in client companies there is a setting at project level that allows you to hide the contact details, this allows them to only see the name of the person.

You can enable this from the settings for the project, from the overview select the cog in the far right, here in the left hand section of the settings page you will find the option to hide the contact details.
