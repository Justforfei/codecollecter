Features & Updates

We’ve made adding tasks easier!

Posted February 26, 2014

This month we’ve introduced a new feature which will speed up your workflow when editing or creating new tasks – shorthand task entry. This feature allows you to quickly create tasks on the Tasks page by entering various shorthand commands, many of which are the same as our already popular reply-by-email commands. For example:

@james @peter [start today] [due tomorrow] Let's complete this task by tomorrow

This will assign the task to James and Peter with a start date of today, a due date of tomorrow and a title of “Let’s complete this task by tomorrow”.

Suggestions

Also, just to make things even quicker for you, we’ve added a dropdown menu that makes suggestions for you as you type. So, for example, if you’re assigning a task to James, you only need to type “@j” and you will be presented with a list of users and companies on the project whose names begin with J. You can then click on their name in the dropdown menu (or press the down arrow on your keyboard and then the enter key) to add them to the task. The list of suggestions will be further refined as you type, and if there is only one suggestion left in the list, pressing the space bar on your keyboard will automatically select the suggestion. If at any time you don’t want to see the suggestions, just press the esc key to hide the dropdown menu. The suggestions work for all of the shorthand task commands, which are outlined below.

Commands

Mentions

@James will assign the task to James. If there is more than one James on the project, you can use the first initial of their surname to pick a specific James, e.g. @James D.

@CompanyX – you can also mention each company that is included on a project. This command will add every employee from CompanyX to the task.

 

Dates

You can set the start date or due date for a task to Today, Tomorrow or next Week. Just put it in square brackets and specify whether it’s the start date or due date, like so:

[Due Next Week] or [start today]

All of the shorthand commands are case insensitive so don’t worry about using capital letters.

 

Priority

You can set the priority on a task using one of three options: ! Low, !! Medium, or !!! High.

 

Notify by email

To notify each assigned user by email, simply type +Notify by email. Simple!

More tips

  • If you enter a date or priority a second time, it will overwrite your previous choice. For example, if you’ve already entered [Due Today], adding [Due Tomorrow] will overwrite the previous due date.
  • If you have not yet added a title for your task, pressing the left and right arrow keys will cycle through the commands you have already entered, highlighting them one by one. Just press the backspace or delete key to remove the selected command.

You may also notice that we’ve made some other general improvements to the performance of the Tasks page just to ensure that your existing tasks load faster than ever, so not only can you create tasks at lightning speed, you can also view them more quickly as well!
