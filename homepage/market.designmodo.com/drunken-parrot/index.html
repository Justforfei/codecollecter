HTML + PSD

The Drunken Parrot UI kit provides an HTML skin of every Bootstrap 3 components. Everything from dropdowns, input groups, alerts, buttons, and even iconography, literally every component have all been re-skinned with a beautiful flat modern style. The kit includes HTML (with CSS, LESS, and other necessary files) and PSD files.
